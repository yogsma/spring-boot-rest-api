# spring-boot-rest-api
How to create REST crud API using spring-boot

# Pre-requisites 
1. Understanding of Java programming
2. Understanding of Spring framework

# What you'll need 
1. Java version 1.8
2. Eclipse Mars.2 
3. Mysql version 5.0 or higher
4. Maven 3.0 or higher
